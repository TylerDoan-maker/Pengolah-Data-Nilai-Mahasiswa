# Pengolah-Data-Nilai-Mahasiswa
Program ini dibuat untuk mengelola dan mengolah data nilai (Tugas, UTS, UAS) mahasiswa dan penghitung rata-rata nilai otomatis.
# Fitur Utama
1. Input Data Mahasiswa: Masukkan jumlah mahasiswa (max 30), NIM, dan nilai (Tugas, UTS, UAS). NIM disimpan di array 1D, nilai di matriks 2D. Menggunakan perulangan for.
2. Tampilkan Data Mahasiswa: Tampilkan data dalam tabel rapi. Menggunakan perulangan untuk iterasi.
3. Ubah Data Mahasiswa: Cari berdasarkan NIM, ubah nilai. Menggunakan perulangan dan strcmp.
4. Hapus Data Mahasiswa: Cari berdasarkan NIM, hapus dengan geser array. Menggunakan perulangan.
5. Hitung Nilai: Hitung rata-rata per komponen dan keseluruhan menggunakan fungsi rekursif hitung_total_rekursif (hasil rekursif ditampilkan). Menggunakan perulangan untuk output per mahasiswa.
# Elemen Teknis
1. Array: 1D untuk NIM, matriks 2D untuk nilai.
2. Perulangan: for untuk iterasi data.
3. Matriks: 3 kolom per mahasiswa.
4. Fungsi Rekursif: Untuk total nilai, hasil ditampilkan di layar.
